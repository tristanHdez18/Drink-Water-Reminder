# Requirements
You need install [plyer] (https://github.com/kivy/plyer) and [Python3] (https://www.python.org/download/releases/3.0/)

```
pip install plyer
```
Finally you can run with the next command:

**Linux:**

```
python3 main.py&
```

**Windows:**
```
pythonw .\main.py
```
